## Crop disease detection and classification 
Developed Convolutional Neural Network with TensorFlow (CNN) and deployed with ReactJs.


### Run on your local machine<br>

To start React server, input in terminal<br>
<code>cd frontend</code><br>
<code>npm run start</code><br>
React server runs on http://localhost:3000/

To run Backend, input<br>
<code>py main.py</code><br>
Python FastAPI (backend) server runs on http://localhost:1242/
